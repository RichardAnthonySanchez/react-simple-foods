# Simple Food Finder

A React application that helps users discover simple food products with 3 ingredients or less, tailored to their dietary preferences and nutritional goals.

## Overview

This simple food finder makes healthy eating easier by focusing on minimally processed foods. Users can filter products based on their allergies, macronutrient goals, and calorie requirements, then build a personalized shopping list that can be exported or printed.

## Features

- **Smart Product Discovery**: Find food products with 3 ingredients or less
- **Personalized Filtering**: Set preferences for:
  - Allergies and dietary restrictions
  - Macronutrient goals (protein, carbs, fats)
  - Calorie requirements
- **Shopping List Management**: Add products to your cart as you browse
- **Export Options**: Print your shopping list or export it for easy shopping

## Tech Stack

- **React** - UI framework
- **Vite 7** - Build tool and development server
- **TypeScript** - Type-safe development
- **Node.js** - Runtime environment

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <your-repo-url>
cd simple-food-finder
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run type-check` - Run TypeScript compiler check

## Usage

1. **Set Your Preferences**: Configure your dietary restrictions, allergies, and nutritional goals
2. **Browse Products**: Explore food items that meet your criteria
3. **Add to Cart**: Build your shopping list by adding products
4. **Export**: Print or export your list when you're ready to shop
